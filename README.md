# 📘 English Learning App

## 📌 Giới thiệu

**English Learning App** là một ứng dụng học tiếng Anh đơn giản được xây dựng bằng Flutter.
Ứng dụng giúp người dùng dễ dàng xem danh sách các bài học và truy cập vào nội dung chi tiết của từng bài.

---

## 🎯 Mục tiêu

* Xây dựng ứng dụng Flutter cơ bản
* Hiển thị danh sách bài học
* Xem chi tiết nội dung từng bài học
* Làm quen với điều hướng màn hình trong Flutter

---

## 🚀 Chức năng chính

### 📋 Trang danh sách bài học

* Hiển thị danh sách các bài học tiếng Anh
* Bao gồm:

  * Tiêu đề
  * Mô tả ngắn
  * Hình ảnh minh họa

### 📘 Trang chi tiết bài học

* Hiển thị nội dung chi tiết của bài học
* Có thể đọc và học trực tiếp

### 👉 Điều hướng

* Click vào bài học → chuyển sang trang chi tiết

## ▶️ Cách chạy ứng dụng

### 🔹 1. Clone project

```bash
git clone https://github.com/20220744-lab/ba-itapcd2.git
cd ba-itapcd2
```

---

### 🔹 2. Cài dependencies

```bash
flutter pub get
```

---

### 🔹 3. Chạy ứng dụng

```bash
flutter run -d chrome
```

*(hoặc chạy trên macOS / Android nếu đã cài môi trường)*
